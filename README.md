# 个人收藏分享网站

这是一个静态 HTML 个人网站原型，用于整理、分类和分享从不同平台收藏的链接与内容。

## 当前功能

- 导入链接、标题和备注
- 按关键词自动分类
- 搜索与筛选收藏内容
- 标记待复核内容
- 切换公开/私密状态
- 导出 CSV 台账

## 打开方式

直接打开：

`outputs/personal-link-site/index.html`

## 主要文件

- `outputs/personal-link-site/index.html`：页面结构
- `outputs/personal-link-site/styles.css`：界面样式
- `outputs/personal-link-site/app.js`：本地交互与自动分类逻辑
